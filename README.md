# Registro de errores técnicos
Este repositorio documenta errores, incidencias y soluciones encontradas durante el despliegue y mantenimiento de distintos proyectos.

Incluye:

- Problemas de infraestructura (VPS, proxy, contenedores)
- Errores en herramientas de automatización
- Incidencias en despliegues
- Diagnóstico y lecciones aprendidas

El objetivo es construir una base de conocimiento técnica y trazable.
